# FPGALinuxVGAGraphics
## Linux on FPGA + VGA graphics 

<h3>Instructions on how to run the c files:</h3> 

Make sure that the header file (.h) is in the same directory as the c file

Compile:
gcc <file_name>.c -o <file_name>

Run:
./<file_name>
